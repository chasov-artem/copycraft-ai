# Відгук на вакансію Product Engineer (Frontend/No-code)

## Проект: CopyCraft AI — SaaS для генерації контенту (ніша нерухомість)

**Посилання на живий демо:** https://copycraft-ai.vercel.app  
**Посилання на код:** https://github.com/chasov-artem/copycraft-ai

## Що реалізовано за 2 тижні (пілотний проект):
- ✅ Лендінг з тарифами та прикладами
- ✅ Повна автентифікація (Firebase Auth + Firestore)
- ✅ Бібліотека шаблонів з фільтрацією за категоріями/тегами
- ✅ Конструктор запитів з динамічними полями
- ✅ Імітація AI-генерації (затримка, скелетони)
- ✅ Повна імітація платіжної системи (checkout, підписки, вебхуки)
- ✅ Адаптивний дизайн, мікроанімації

## Відповідність вимогам вакансії:
| Вимога | Реалізація |
|--------|------------|
| SaaS-інтерфейси | Кабінет користувача, білінг, бібліотека шаблонів |
| Робота з шаблонами/промптами | Динамічні форми, теги, категорії |
| Білінг | Mock Stripe/Paddle з повним циклом |
| Next.js + UI Kit | Next.js 14 + Shadcn/ui |
| Швидкість та результат | MVP за 2 тижні |

## Чому я ідеальний кандидат:
- **Продуктове мислення:** я не просто написав код, я спроектував архітектуру, яку легко масштабувати.
- **Адаптивність:** через неможливість використовувати Stripe в Україні, я реалізував повноцінну імітацію, яка демонструє розуміння всієї системи платежів.
- **Швидкість:** проект повністю готовий за 2 тижні, включаючи всі ключові функції.
- **Досвід з Firebase:** успішна інтеграція Auth, Firestore, security rules.
